# ComfortLine brand assets

Version: 2026.1

Use `comfortline-logo-primary.svg` on charcoal or dark photography and
`comfortline-logo-on-light.svg` on white or pale backgrounds. SVG is the
preferred digital master. Use PNG when transparency is required, JPEG only on
the supplied fixed background, WebP for compact web delivery, and PDF/EPS for
professional print workflows.

Maintain clear space equal to one wordmark cap height on every side. Minimum
digital width is 120 px for the full lockup and 24 px for the symbol. Do not
stretch, rotate, recolor, add effects, rearrange, or place the logo over a busy
background.

Core colors:

- Charcoal: `#0E0D13`
- Signature Gold: `#B59C73`
- Warm White: `#F5F0E8`
- Paper: `#F8F7F5`
- Ink: `#131218`

Typography: Montserrat, with Arial and the system sans-serif stack as fallbacks.

The downloadable token file follows the W3C Design Tokens Community Group
format. Interface text and controls should meet WCAG 2.2 AA contrast.
